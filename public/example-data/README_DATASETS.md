# ML Playground example data

Files:
- retail_sales_forecasting.csv → regression dataset for sales forecasting / revenue prediction
- saas_churn_classification.csv → classification dataset for churn prediction
- retail_ml_playground.db → SQLite example database for retail analytics
- saas_ml_playground.db → SQLite example database for SaaS churn analytics
- retail_ml_playground.sql → schema script for the retail DB
- saas_ml_playground.sql → schema script for the SaaS DB

Suggested targets:
- Retail: units_sold or revenue
- SaaS: churn
